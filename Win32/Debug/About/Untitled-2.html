HAVANA CRM

СИСТЕМА УЧЕТА КЛИЕНТОВ ДЛЯ ФОТОГРАФОВ

Добро пожаловать в первую распределенную систему учета клиентов для фотографов и визажистов. Система изначально создавалась как учебный продукт и как персональное средство. Но после создания и внедрения системы было принято решение предоставить систему для коолективного использования.

Система была создана одним человеком за год. Данный конкретный продукт - это Бета-версия 1.0. Havana-CRM так же обеспечивает доступ с WEB-интерфейса и позволяет добавлять клентов из приложения для ОС Android.

ПОЯВИТСЯ В СЛЕДУЮЩЕЙ ВЕРСИИ

WINDOWS ВЕРСИЯ

  • Проверка доступа в Интернет;
  • Клавиатурные комбинации;
  • Фото клиентов;
  • Фото пользователя и возможность правки личной информации;
  • Дополнительные поля: пол, папка на компьютере, одиночный статус для заказов и клиентов и т.д.;
  • Рассылки;
  • Подарочные сертификаты;
  • Несколько клиентов в 1 организации;
  • Обновленный интерфейс;

ANDROID ВЕРСИЯ

  • Добавление, удаление, просмотр, правка клиентов;
  • Добавление, удаление, просмотр, правка заказов;
  • Возможность выбрать звонки или смс, на основании которых будет создан клиент;

WEB ВЕРСИЯ

  • Добавление, удаление, редактирование, просмотр заказов;
  • Отчеты и статистика;
  • Плагин Контактная форма для Wordpress;
  • Плагин "Статус заказов" для Wordpress;

ТЕКУЩИЕ ВОЗМОЖНОСТИ СИСТЕМЫ

WINDOWS ВЕРСИЯ

  • Аутентификация;
  • Добавление, удаление, редактирование, просмотр клиентов;
  • Добавление, удаление, редактирование, просмотр заказов;
  • Стастика по заказам за год и месяц;
  • Картина текущих заказов;
  • Графическое представление отчетов (диаграммы);
  • Мониторинг "зависших" заказов;

ANDROID ВЕРСИЯ

  • Аутентификация;
  • Добавление последнего звонившего в качестве клиента в систему;

WEB ВЕРСИЯ

  • Аутентификация;
  • Добавление, удаление, редактирование, просмотр клиентов;
  • Картина текущих заказов;
